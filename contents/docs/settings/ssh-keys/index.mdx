When deploying infrastructure using Forge, SSH keys are required to authenticate with remote servers. These keys are used to manage the infrastructure, and also to provide users with secure access to their own resources.

## Creating SSH Keys
To create a new SSH Key within Forge, simply follow the steps below.
1. Select the menu button located at the bottom of the application sidebar.
2. Select "Settings" from the menu.
3. Select "SSH Keys" from the settings navigation.
4. Click the "Create Key" button.
5. Complete the form and download the key when prompted.

<Note type="warning" title="Warning">
    Don&apos;t forget to download your SSH key. Once you close the modal, there is no way to download it again.
</Note>
