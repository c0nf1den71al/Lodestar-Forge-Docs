Any domains which are utilised for your red team projects can be added to Forge through the "domains" section. Added domains can be used later when deploying and configuring infrastructure and can be tracked for signs of triage.

## Domain Details

Each domain within Forge has several attributes, each of which are outlined below.

* **Domain:** The domain added to Forge.
* **Domain State:** The state of the domain, one of the following: "Pending Analysis", "Healthy", "Burnt", "Aging", "Unhealthy", "Archived".
* **Autoscan:** Whether the domain is automatically scanned for signs of triage.
* **Description:** A description of the domain.
